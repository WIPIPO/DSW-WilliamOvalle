Que son las etiquetas, atributos y valores de HTML

Este sitio web pretende orientar a los interesados de entender que son las etiquetas, los atributos y valores de HTML, como también se ilustrara el eso de las etiquetas como su apertura, cierre y que contiene dentro de estas etiquetas, como sabemos estas etiquetas es muy importante en el lenguaje de HTML para la construcción de sitios WEB.

Temas que se trataran

Que son las etiquetas

.cuales son las etiquetas mas usadas


Etiqueta Descripción


!DOCTYPE tipo de documento (versión de html empleada)
html engloba todo el documento
comentario (sólo visible en el código fuente)

Metadatos


Etiqueta Descripción
head delimita el encabezado del documento
title título del documento (se muestra en la pestaña del navegador)
base URI base para direcciones relativas
link enlace a otros archivos (hoja de estilo, etc.)
meta metainformación sobre el documento
style hoja de estilo incluida en el documento

Secciones


Etiqueta Descripción
body delimita el cuerpo del documento
article artículo
section sección
nav navegación
aside lateral
h1 a h6 encabezado (de nivel 1 a 6)
hgroup grupo de encabezados
header cabecera
footer pie
address dirección (información sobre el autor)

Bloques de contenido


Etiqueta Descripción
p párrafo
hr separador
div división
blockquote cita larga (que incluye varios párrafos)
pre texto preformateado
main principal
figure ilustración
figcaption pie de ilustración

Texto (en línea)


Etiqueta Descripción
br salto de línea
wbr posible salto de línea
a hiperenlace
span contenedor de texto genérico
strong importante
em énfasis
sub subíndice
sup superíndice
abbr abreviatura
b atención
cite obra
data datos
dfn definición
kbd teclado
i tecnicismo
mark resaltado añadido posteriormente, no en el original
q cita
s incorrecto
small comentario
time fecha y hora
u sonido inarticulado
code código (de programa de ordenador)
samp salida (de programa de ordenador)
var variable (de programa de ordenador)
bdo dirección de escritura
bdi ignorar dirección de escritura

Modificaciones


Etiqueta Descripción
ins texto insertado
del texto borrado

Contenido incrustado


Etiqueta Descripción
picture imagen múltiple para diferentes resoluciones o densidades
img imagen
iframe marco incrustado en el documento
embed
object objeto
param parámetro para la etiqueta objeto objeto>
video
audio
source origen de las etiquetas img>, audio> o video>
track
map mapa de imagen
area área en mapa de imagen

Listas (bloque de contenido)


Etiqueta Descripción
ol lista ordenada
ul lista no ordenada
menu equivale a ul
li elemento de lista (ordenada o no ordenada)
dl lista de definición
dt término en lista de definición
dd definición en lista de definición

Tablas


Etiqueta Descripción
table tabla
caption leyenda de tabla
colgroup grupo de columnas
tbody cuerpo de tabla (grupo de filas)
thead cabecera de tabla (grupo de filas)
tfoot pie de tabla (grupo de filas)
tr fila
td celda
th celda de cabecera
col columna

Formularios


Etiqueta Descripción
form formulario
label etiqueta de un control
input control (hay varios tipos)
button botón
select caja de lista
datalist
optgroup grupo de opciones en una caja de lista
option opción de caja de lista
textarea área de texto
output cálculo
progress barra de progreso
meter indicador
fieldset grupo de controles
legend leyenda de grupo de controles

Elementos interactivos


Etiqueta Descripción
details desplegable
summary leyenda para la etiqueta details>
dialog cuadro de diálogo de una aplicación

Scripts


Etiqueta Descripción
script script
noscript contenido a mostrar en navegadores que no admiten etiqueta script>
template plantillas utilizables por scripts
slot marcador para componente
canvas zona de dibujo utilizable por script

Notación Ruby (texto en línea)


Etiqueta Descripción
ruby notación ruby
rt elemento de notación ruby (texto)
rp elemento de notación ruby (paréntesis)
