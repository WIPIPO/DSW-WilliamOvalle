TIPOS DE ETIQUETAS

En HTML encontramos numerosas etiquetas es por eso que en este apartado las vamos a clasificar de las que comúnmente se usan en la creación de páginas web.

Etiquetas Preámbulo y Elemento Raíz

Estas etiquetas se encuentran al inicio de un documento html y hasta en estas clases de etiquetas entra la etiqueta de comentario.

Etiquetas de Metadatos

Estas etiquetas además como su nombre lo dice de tener información meta información de un documento existe dentro de esta clase las etiquetas head, title,base,link,meta,style.

Las Etiquetas de Secciones

son las partes temáticas en las que se puede dividir el contenido de la pagina anterior mente en html4 se usaba para realizar esas divisiones las etiquetas div, en html5 se cambio div ya no se utiliza tanto como en la versión anterior ahora existen las etiquetas body,article,nav,aside,h1 a h6, hgroup, footer, addres.

Las Etiquetas de Bloques De Contenido

estas etiquetas permiten definir bloques donde se muestra por separado de las demás etiquetas bloques en ellas se pueden escribir parágrafos formados por texto e imágenes en este apartado encontramos las etiquetas p, hr, div, blockquote, pre, main, figure y figcaption.

Las Etiquetas De Texto

en línea son las más numerosas y se emplean para identificar fragmentos de texto que tienen un significado especial. Normalmente, los elementos de texto en línea se encuentran dentro de elementos de bloque, modificaciones o tablas, en esta clasificación se encuentran las etiquetas br, wbr, a, span, strong, em, sub, abbr y entre otras que se trataran en el apartado de las descripción de las etiquetas.

Las Etiquetas de Modificaciones

Esta categoría incluye únicamente dos etiquetas, del y ins, pensadas para identificar las modificaciones que se han realizado en un texto.

Las etiquetas de CONTENIDO INCRUSTADO

En esta categoría como su nombre los dice son etiquetas que contienen contenido incrustado en las paginas web como imágenes, audio etc en las que encuentran algunas etiquetas como picture, img, iframe, embed, object, param, video, source y entre otras que se describirán mas adelante.

Las etiquetas de Listas (Bloque De Contenido)

Estas etiquetas son bloques de contenido en las paginas web como listas ordenas o no ordenadas elementos de las mismas listas estas se usan mayor parte a la creación menús y menús desplegables dentro de estas se encuentran las etiquetas ol, ul, menú, li dl, dt, dd.

Las etiquetas de TABLAS

Se utilizan para mostrar la información en formato de tablas y tienen sus etiquetas que definen o modelan la tabla como son las etiquetas table, caption, colgrup, tbody, thead, tfoot, tr, td, th, col.

Las etiquetas de formularios

Una página web puede contener formularios, que permiten al usuario proporcionar información. Normalmente dicha información se envía a un servidor para ser procesada por un programa, aunque también puede ser procesada en el navegador mediante un programa escrito en el lenguaje JavaScript Las etiquetas relacionadas con formularios son: form, fieldset , label, input, button , textarea y select.

Etiquetas de Elementos Interactivos

Los elementos HTML son normalmente elementos fijos y su posible interactividad se consigue mediante JavaScript, pero existen un par de elementos que permiten cierta interactividad (muy limitada) sin necesidad de JavaScript en estas se encuentra las etiquetas details, summary, dialog.

Etiquetas de SCRIPTS

Los navegadores son capaces de ejecutar programas escritos en el lenguaje de programación JavaScript. Estos programas se suelen denominar scripts. Varias etiquetas permiten integrar estos programas en una página web en estas se encuentran las etiquetas script, noscript, template, slot, canvas.
